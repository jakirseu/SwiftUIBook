**বইটি থেকে আপনি কী শিখবেন:**
-   SwiftUI-এর মৌলিক বিষয় এবং লেআউট সিস্টেম।
-   ডেটা ম্যানেজমেন্ট ও স্টেট ম্যানেজমেন্ট।
